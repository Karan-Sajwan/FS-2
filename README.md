# Experiment 1 – Basic SPA using React & Vite

## Description
This project is a Single Page Application (SPA) built using React and Vite as part of Full Stack–II lab experiment.

## Features
- Client-side routing using React Router
- Reusable components
- State management with React Hooks
- API integration using fetch
- Contact form handling

## Tech Stack
- React
- Vite
- JavaScript
- HTML5
- CSS3

## How to Run
```bash
npm install
npm run dev
